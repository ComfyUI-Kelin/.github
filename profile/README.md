# ComfyUI-Kelin

ComfyUI-Kelin is an organization focused on building practical, well-maintained
custom nodes and workflow tools for ComfyUI.

这里主要维护与 ComfyUI 相关的自定义节点、工作流工具，以及围绕图像生成、
抠图后处理、远程调用和 LLM 能力集成的实用扩展。

## Focus

- ComfyUI custom nodes for real production workflows
- Image generation and post-processing utilities
- LLM-assisted workflow components
- Remote service integration for ComfyUI

## Public Repositories
